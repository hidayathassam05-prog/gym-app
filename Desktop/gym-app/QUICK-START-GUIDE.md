# 🏃 FitTrack Quick Start Guide

**Welcome to FitTrack!** Your personal fitness management system.

---

## 📋 Getting Started (5 Minutes)

### 1. Create Your Account

1. Open `index.html` in your browser
2. Click **"Sign Up"** tab
3. Enter your details:
   - **Full Name**: Your name
   - **Email**: Your email address
   - **Password**: Min 8 chars with uppercase, lowercase, digit, special char
   - **Confirm Password**: Re-enter password
   - **Role**: Select your role:
     - 👤 **Member** - Track personal fitness
     - 👨‍🏫 **Trainer** - Manage classes & members
     - 🔐 **Administrator** - Full system access

4. Click **"Create Account"**

### 2. Login

1. Enter your email and password
2. Click **"Login"**
3. You're now in your **Dashboard**!

## 🎯 Features by Role

### 👤 Member Dashboard

#### Dashboard
View your fitness statistics:
- 💪 Workouts this week
- 📊 Total sessions completed
- 🔥 Calories burned
- 🎯 Current workout streak

#### Log Workout
1. Go to **Workouts** section
2. Select exercise type (Chest, Legs, Back, etc.)
3. Enter:
   - Duration (minutes)
   - Sets
   - Reps per set
   - Calories burned
4. Click **"✓ Log Workout"**

#### Track Health
1. Go to **Health** section

**💧 Daily Water Intake**
- Enter amount in ml (250ml = 1 cup)
- Track progress toward 2L goal
- Visual progress bar shows percentage

**📏 Body Measurements**
- Select measurement type (Weight, Chest, Waist, Arms, Thighs)
- Enter value
- Click **"📏 Log Measurement"**
- View historical tracking

#### Set Fitness Goals
1. Go to **Goals** section
2. Enter:
   - Goal name (e.g., "Lose 10kg")
   - Target value (e.g., "80kg")
   - Deadline date
3. Click **"🎯 Create Goal"**
4. Mark complete when achieved

#### View Profile
1. Go to **Profile** section
2. Upload profile picture (click avatar)
3. View achievements earned
4. See performance statistics

#### Join Classes
1. Go to **Classes** section
2. View available fitness classes
3. Click **"Join"** to register

### 👨‍🏫 Trainer Dashboard

All Member features plus:
- **Classes**: Create and manage fitness classes
- **Clients**: View assigned member workouts
- **Analytics**: Track member progress

### 🔐 Admin Dashboard

Full system access:
- **Members**: Manage all user accounts
- **Classes**: Create and schedule classes
- **Analytics**: View gym-wide statistics
- **Users**: Manage trainers and members

## 🐛 Troubleshooting

### Issue: "Your session has expired"
- You've been inactive for 30 minutes
- **Solution**: Login again

### Issue: "Account is locked"
- You've had 5 failed login attempts
- **Solution**: Wait 15 minutes and try again

### Issue: Data not saving
- Check browser's localStorage is enabled
- **Solution**: Clear cache and refresh page

### Issue: Can't see all sections
- Check your user role
- **Solution**: Different roles have different features

### Issue: Password too weak
- Need more character variety
- **Solution**: Add uppercase, lowercase, numbers, and special chars

## 💾 Data & Privacy

### Where is My Data Stored?
- Your data is stored in your **browser's localStorage**
- Not synced to cloud (currently)
- Data persists across browser sessions
- Only visible to your account

### What Data is Collected?
- ✅ Your name and email
- ✅ Fitness activities (workouts, goals, measurements)
- ✅ Health metrics (water intake)
- ✅ Profile information
- ✅ Login activity logs

### Data Security
- 🔒 Password protected
- 🔒 Role-based access control
- 🔒 Session validation
- ⚠️ Not encrypted (future improvement)

## 🎓 Best Practices

### Effective Goal Setting
1. **Specific** - "Lose 10kg" not "Get healthier"
2. **Measurable** - Track with numbers
3. **Achievable** - Realistic timeframe
4. **Relevant** - Important to you
5. **Time-bound** - Set a deadline

### Workout Logging
- Log **immediately** after exercise (while fresh)
- Be **accurate** with durations and reps
- Include **all** exercises (not just main ones)
- Use **consistent** exercise names

### Health Tracking
- Measure at **same time of day**
- Measure **weekly** for accuracy
- Drink water **consistently** throughout day
- Track **one measurement** at a time for focus
